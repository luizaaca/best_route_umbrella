# Repositório Umbrella: Best Route

Este repositório serve apenas como referência e documentação para os dois projetos principais:

- [**api_best_route** (Backend Python)](https://github.com/luizaaca/api_best_route)
- [**best_route_tsp_front** (Frontend JavaScript)](https://github.com/luizaaca/best_route_tsp_front)

## Como rodar localmente

Clone ambos os repositórios:

```sh
git clone https://github.com/luizaaca/api_best_route
git clone https://github.com/luizaaca/best_route_tsp_front
```

Siga as instruções de cada projeto para rodar localmente o backend e o frontend.
